HTML and CSS

HTML and CSS are the basic building blocks of web development. HTML is used to structure content, while CSS is used for styling.

HTML stands for HyperText Markup Language and provides the basic structure of a webpage.

CSS, or Cascading Style Sheets, controls the layout and presentation of elements on the page.

<!DOCTYPE html> <html> <head></head> <body></body> <link rel="stylesheet" href="styles.css">
  • HTML tags structure content
  • CSS controls layout
  • Use CSS classes to style HTML elements
  • HTML5 introduced new semantic elements
  • CSS Flexbox is a powerful layout tool

JavaScript and Java

JavaScript is a dynamic programming language used to create interactive effects within web browsers.

Java is a general-purpose programming language that is class-based and object-oriented.

While JavaScript is primarily used for client-side development, Java is often used for server-side applications.

const message = "Hello, World!"; function greet() { alert("Hello!"); } class Car { constructor(make, model) { this.make = make; this.model = model; } } let x = 5; console.log(message);
  • JavaScript is event-driven
  • Java is platform-independent
  • JavaScript can manipulate DOM elements
  • Java is used in enterprise applications
  • Both JavaScript and Java are object-oriented

Python and Ruby

Python is a high-level programming language known for its readability and versatility.

Ruby is an interpreted language focused on simplicity and productivity, often used in web development.

Both languages are dynamic and object-oriented, with Python being used for a variety of applications from data science to web development.

def greet(name): print(f"Hello, {name}!") class Person: def __init__(self, name): self.name = name
  • Python is great for machine learning
  • Ruby on Rails is a popular framework
  • Python has extensive libraries for data analysis
  • Ruby is known for its elegant syntax
  • Both support functional and object-oriented paradigms

APIs and Data

APIs (Application Programming Interfaces) allow different software systems to communicate with each other.

APIs are used to interact with databases, retrieve data, and send data back to a server or client.

Data is often exchanged in JSON or XML formats through API requests.

fetch('/data').then(response => response.json()).then(data => console.log(data)); const apiUrl = "https://api.example.com/data"; $.ajax({ url: apiUrl, type: "GET", success: function(response) { console.log(response); } }); const data = { name: "John", age: 30 }; const headers = { "Content-Type": "application/json" };
  • REST APIs use HTTP requests
  • APIs often return data in JSON format
  • GET and POST are the most common HTTP methods
  • Authentication is important for secure APIs
  • APIs can be public or private

Frameworks and Libraries

Frameworks and libraries are tools that help developers build applications faster and more efficiently.

Frameworks usually provide a structure for an app, while libraries offer reusable functions or components.

Popular frameworks include React, Angular, and Django, while libraries like Lodash and jQuery help with specific tasks.

import React from 'react'; import { useState } from 'react'; const app = express(); app.get('/', (req, res) => res.send('Hello World')); import pandas as pd;
  • React is a popular JavaScript library for building UIs
  • Angular is a comprehensive framework for front-end development
  • Django is a Python framework for web development
  • jQuery simplifies DOM manipulation
  • Vue.js is a progressive JavaScript framework
